\begin{abstract}
Czech and Slovak are languages which don't exhibit a manifest position
for the Articles in the Determiner Phrase. The aim of this paper is to
show how this structure is accessed during the learning of Italian, a
language which presents the articles as for the standard behavior for
nouns.\par \textbf{Keywords:} Determiner Phrase, Italian L2, Second
Language Acquisition, Syntax
\end{abstract}

\section{Introduction}\label{introduction}

Czech (\texttt{cs}) and Slovak (\texttt{sk}) are languages of the Slavic
branch in the Indo-european family. Alongside a certain morphological
complexity in noun declension systems, these languages -- except for
Bulgarian and Macedonian \citep[ in \citet{wals}]{wals-37} -- don't show
an overt realization of the Determiner position inside the noun phrase
\citep{harkins1953}. Conversly, Italian (\texttt{it}) and the other
romance languages explicit that position as a default behaviour, usually
with a free morpheme before the noun\footnote{Otherwise, Romanian uses a
  intralexematic suffixation for the article, which falls in the end of
  the word.}:

\begin{enumerate}
\def\labelenumi{(\arabic{enumi})}
\tightlist
\item
  Slavic languages

  \begin{enumerate}
  \def\labelenumii{\alph{enumii}.}
  \tightlist
  \item
    (\texttt{cs}, \emph{Reference})\\
    \gll    Viděl jsem Ø/?ten chlapec který běží\\
    See.PTCP.PST AUX.1sg DEM Boy.SG pro Run.3sg\\
    \glt        
  \end{enumerate}
\item
  Romance languages

  \begin{enumerate}
  \def\labelenumii{\alph{enumii}.}
  \tightlist
  \item
    (\texttt{it}, \emph{Reference})\\
    \gll    Ho visto un/il/*Ø ragazzo che corre\\
    AUX.1sg See.PTCP.PST DET Boy.SG pro Run.3sg\\
    \glt  
  \item
    (\texttt{fr}, \emph{Reference})\\
    \gll    J'ai vu un/le/?Ø garçon courir\\
    I.AUX.1sg See.PTCP.PST DET Boy.SG Run\\
    \glt  
  \end{enumerate}
\end{enumerate}
